# La Clave Chat

Plataforma de mensajeria propia basada en la base tecnica de Telegram Web K, con identidad y branding 100% personalizado de **La Clave Argentina**.

## Identidad
- **Nombre:** La Clave Chat
- **Color de marca:** `#0aa3ff` (celeste) / `#0a0a0a` (negro, fondo)
- **Emoji / Logo:** 🚀 (placeholder hasta subir uno definitivo)

## Despliegue
- **Repositorio:** https://github.com/florenciaamor36-ship-it/tlgmlcvrgtna
- **App publica:** https://florenciaamor36-ship-it.github.io/tlgmlcvrgtna/
- **Hosting:** GitHub Pages (rama `main`)

## Base tecnica
- **Origen:** [morethanwords/tweb](https://github.com/morethanwords/tweb) (Telegram Web K)
- **Arquitectura:** Frontend estatico (HTML / JS / CSS), MTProto para mensajeria
- **Total de archivos:** 1.641 (~61 MB)
